## 1. 科技未來主義 (Cyberpunk/Futuristic)

### Design Movement
**Cyberpunk / High-Tech Industrial**
結合工業監控的實用性與未來科技的視覺衝擊，打造如同科幻電影中的指揮中心。

### Core Principles
1. **資訊高密度與層次感**：在單一畫面中呈現大量數據，透過顏色與亮度區分層次，確保關鍵資訊突出。
2. **暗色模式為核心**：以深色背景減少螢幕眩光，適合長時間監控，並讓彩色數據更鮮明。
3. **動態數據視覺化**：數據不是靜止的，而是流動的、呼吸的，透過動畫展現系統的生命力。
4. **模組化與網格系統**：嚴謹的網格佈局，每個模組如同積木般精密契合，展現工程的嚴謹性。

### Color Philosophy
- **背景**：深灰藍 (Dark Slate Blue) 與 炭黑 (Charcoal)，營造深邃的空間感。
- **主色**：霓虹青 (Neon Cyan) 代表正常運作與水流，充滿科技感與活力。
- **警示色**：螢光橘 (Fluorescent Orange) 與 警示紅 (Alert Red)，在暗色背景中極具穿透力。
- **輔助色**：冷鋼灰 (Cool Steel Grey) 用於結構線條與次要資訊。
- **意涵**：深色背景象徵未知的挑戰（暴雨、黑夜），亮色數據象徵科技帶來的掌控與光明。

### Layout Paradigm
**Bento Box Grid (便當盒網格) + HUD (抬頭顯示器)**
- 捨棄傳統的由上而下網頁佈局，採用全螢幕儀表板設計。
- 畫面被分割為多個功能區塊（Bento Box），每個區塊獨立運作但視覺風格統一。
- 關鍵數據（如水位、警戒狀態）採用 HUD 風格，懸浮於地圖或模型之上。

### Signature Elements
1. **玻璃擬態 (Glassmorphism) 面板**：半透明的深色玻璃背景，帶有模糊效果與細緻邊框，增加介面的通透感與現代感。
2. **數據流光 (Data Streams)**：連接各模組的線條帶有流動的光效，象徵數據的即時傳輸。
3. **科技感圖標與字體**：使用單色線條圖標，搭配等寬字體 (Monospace) 顯示數值，強化工程屬性。

### Interaction Philosophy
- **即時反饋**：滑鼠懸停時，模組邊框發光或數據放大，提供明確的互動回饋。
- **平滑過渡**：數據更新時不突兀跳動，而是平滑滾動或漸變。
- **深度導航**：點擊模組可展開詳細資訊層，如同深入系統內部。

### Animation
- **呼吸燈效**：正常運作的設備有緩慢的呼吸燈效，異常時轉為急促閃爍。
- **數據加載**：圖表繪製採用掃描線或逐步填充動畫。
- **警報觸發**：全螢幕邊緣紅光脈衝，強烈吸引注意力。

### Typography System
- **標題**：`Orbitron` 或 `Rajdhani` (Google Fonts)，幾何感強，帶有科技風格。
- **內文**：`Inter` 或 `Roboto`，保持高可讀性。
- **數據**：`JetBrains Mono` 或 `Fira Code`，等寬字體確保數值對齊與易讀。

## 2. 極簡工業風 (Minimalist Industrial)

### Design Movement
**Swiss Style / International Typographic Style**
強調客觀、清晰與秩序，去除多餘裝飾，回歸數據本質。

### Core Principles
1. **形式追隨功能**：所有設計元素都必須有明確的功能目的，無裝飾性圖案。
2. **極致的清晰度**：透過大量的留白與嚴格的排版，讓資訊一目了然。
3. **中性色彩**：避免情緒化的色彩，以中性色調為主，僅在必要時使用色彩標示狀態。
4. **結構化佈局**：使用不對稱但平衡的佈局，引導使用者的視線流動。

### Color Philosophy
- **背景**：淺灰 (Light Grey) 或 純白 (White)，提供乾淨的畫布。
- **文字**：深灰 (Dark Grey) 與 黑色 (Black)，高對比度確保閱讀性。
- **狀態色**：使用低飽和度的 藍 (Blue)、黃 (Yellow)、紅 (Red) 指示狀態，不刺眼但明確。
- **意涵**：冷靜、客觀、專業，強調數據的真實性與可信度。

### Layout Paradigm
**Modular Grid System**
- 嚴格的欄位網格系統，資訊模組依據重要性佔據不同比例的空間。
- 運用大量的留白 (Whitespace) 來區隔資訊，而非線條或邊框。

### Signature Elements
1. **粗線條與幾何圖形**：使用粗黑線條區隔主要區塊，搭配簡單的幾何圖形（圓、方）表示設備。
2. **大字級排版**：關鍵數據（如水位）使用超大字級，成為視覺焦點。
3. **扁平化設計**：完全去除陰影、漸層與立體感，追求極致的平面化。

### Interaction Philosophy
- **微互動**：按鈕與連結有細微的顏色變化或位移。
- **資訊揭露**：預設僅顯示最關鍵資訊，滑鼠懸停或點擊後才顯示細節。

### Animation
- **極簡轉場**：頁面切換採用快速的淡入淡出或滑動。
- **數據變化**：數值直接跳轉，不使用過多的過渡動畫。

### Typography System
- **全站字體**：`Helvetica Now` 或 `Arial`，經典、中性、無襯線。
- **層級**：透過字重 (Bold/Light) 與字級大小建立強烈的視覺層級。

## 3. 生態水文美學 (Eco-Hydrological Aesthetic)

### Design Movement
**Organic Modernism / Biophilic Design**
將自然的流動感與現代設計結合，強調水利工程與環境的共生。

### Core Principles
1. **流動性**：設計元素模仿水的流動與波紋，打破僵硬的直線。
2. **自然色調**：從自然界提取色彩，營造和諧、安心的氛圍。
3. **柔和的視覺語言**：使用圓角、曲線與漸層，減少工業設施的冰冷感。
4. **環境數據整合**：不僅關注設備，更強調天氣、水文與環境的關聯。

### Color Philosophy
- **背景**：淺水藍 (Pale Aqua) 或 霧白 (Mist White)。
- **主色**：深海藍 (Deep Sea Blue) 與 湖水綠 (Teal)。
- **輔助色**：沙色 (Sand) 與 雲灰 (Cloud Grey)。
- **警示色**：珊瑚紅 (Coral Red)，溫暖但醒目。
- **意涵**：強調水資源的管理與保護，而非單純的對抗與控制。

### Layout Paradigm
**Fluid Layout**
- 區塊之間沒有明顯的邊界，透過背景色的漸層或留白自然過渡。
- 佈局如同河流般蜿蜒，引導視線順暢流動。

### Signature Elements
1. **波浪曲線**：用於圖表背景、分隔線或裝飾圖案。
2. **磨砂玻璃與水滴效果**：模仿水氣與玻璃的質感。
3. **實景影像融合**：大量使用高品質的現場照片或即時影像作為背景或視覺中心。

### Interaction Philosophy
- **彈性回饋**：互動帶有彈性 (Elasticity)，如同觸碰水面。
- **視差滾動**：背景與前景元素以不同速度移動，營造深度感。

### Animation
- **液態動畫**：圖表數據變化如水位升降般平滑流動。
- **波紋效果**：點擊或警報觸發時產生擴散的波紋動畫。

### Typography System
- **標題**：`Montserrat` 或 `Nunito`，圓潤、親和力強。
- **內文**：`Lato` 或 `Open Sans`，清晰易讀。

## Selected Design Philosophy: 科技未來主義 (Cyberpunk/Futuristic)

我們選擇 **科技未來主義** 作為中山抽水站智慧戰情室的設計風格。

**理由：**
1. **戰情室的本質**：戰情室是用於監控、決策與指揮的核心場所，需要展現高度的專業性、科技感與掌控力。科技未來主義的風格完美契合這一需求。
2. **數據可視化需求**：面對大量且複雜的即時數據（水位、泵浦狀態、預警），深色背景與高對比度的霓虹色彩能提供最佳的閱讀體驗，並能有效區分正常與警戒狀態。
3. **預警與應變**：此風格強調的動態視覺效果與強烈的警示色，能在緊急狀況發生時迅速抓住操作人員的注意力，縮短反應時間。
4. **創新形象**：採用此風格能突顯中山抽水站導入 AI 與現代化監控技術的創新形象，擺脫傳統公共工程沈悶、過時的刻板印象。

**執行重點：**
- **深色主題**：全站採用深灰藍/炭黑背景。
- **霓虹點綴**：使用青色、亮橘色作為關鍵數據與互動元素的強調色。
- **玻璃擬態**：運用半透明面板承載資訊，增加介面層次。
- **動態數據**：確保所有圖表與數值都具備流暢的動態效果。
- **字體選擇**：標題使用 `Orbitron`，數據使用 `JetBrains Mono`。
